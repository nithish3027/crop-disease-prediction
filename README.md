# CropAssist – AI Crop Disease Assistant

CropAssist is an AI-powered web application that helps farmers detect crop diseases early by analyzing plant leaf images. The system provides quick predictions and guidance to improve crop health and productivity.

## Features

- 🌿 AI-based leaf disease detection
- 🔍 Upload crop leaf images for instant analysis
- 💧 Fertilizer recommendation system
- 📢 Farming advisory alerts
- ⚙️ Tools and maintenance guide for farmers
- 📆 Seasonal crop planning support
- 🌍 Multi-language translation support

## Technologies Used

- HTML
- CSS
- JavaScript
- AI Model (ResNet50)
- Hugging Face Spaces
- Service Workers (PWA)

## How It Works

1. Upload a crop leaf image
2. AI analyzes the image
3. Disease prediction is generated
4. Farmers receive guidance and recommendations
